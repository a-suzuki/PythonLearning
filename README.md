## Emoji チートシート

コミットログでは、以下の該当する絵文字を利用してみよー

* :new: `:new:` 新しい機能追加
* :bug: `:bug:` バグ修正
* :memo: `:memo:` ドキュメント更新
* :art: `:art:` リファクタリング
* :fire: `:fire:` コードやファイル削除
* :racehorse: `:racehorse:` パフォーマンス改善
* :white_check_mark: `:white_check_mark:` テスト追加
* :green_heart: `:green_heart:` CIビルド修正
* :crystal_ball: `:crystal_ball:` 実験
* :shipit: `:shipit:` リリース
